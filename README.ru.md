## О EnvStudio

**EnvStudio** — это современный менеджер переменных среды Windows, построенный на WinUI 3. Он заменяет 20-летний диалог «Свойства системы → Переменные среды» нативным интерфейсом Windows 11.

## Основные возможности

- **Интуитивный визуальный редактор** — Добавляйте, редактируйте, удаляйте и переупорядочивайте переменные среды в современном интерфейсе WinUI 3.
- **Управление списком PATH** — Перетаскивание для сортировки, обнаружение дубликатов и неработающих ссылок.
- **Обнаружение конфликтов EXE** — Сканирует все каталоги в PATH на наличие исполняемых файлов и показывает, какие переопределены более ранними записями, с переходом к конфликтующей записи в один клик.
- **Обнаружение конфликтов области видимости** — Когда пользовательская переменная затеняет системную с тем же именем, показывает зачёркивание с иконкой предупреждения, чтобы вы всегда знали, какое значение реально действует.
- **Неразрушающий переключатель** — Отключайте переменную без удаления. Молча переименовывает значение реестра, сохраняя оригинальные данные. Повторно активируйте в любой момент одним кликом.
- **Проверки безопасности при удалении** — Перед удалением переменной сканирует все остальные переменные на наличие ссылок `%VARNAME%` и предупреждает о возможных поломках.
- **Предпросмотр раскрытия переменных** — Наведите курсор на любое значение, содержащее ссылки `%VAR%`, чтобы увидеть полностью раскрытый путь во всплывающей подсказке.
- **Обнаружение внешних изменений** — Отслеживает реестр в реальном времени. Если другая программа изменяет переменные среды, пока EnvStudio открыт, вы получаете уведомление.
- **Система профилей** — Сохраняйте, переключайте и применяйте профили переменных для разных сред разработки (например, «Java 8», «Node.js», «AI/ML»).
- **Снимки и откат** — Автоматические снимки перед каждым изменением, с diff-просмотром в стиле Git и откатом в один клик.
- **Поиск и фильтрация** — Быстрый поиск переменных в областях пользователя и системы с поддержкой регулярных выражений. Совпадающие подзаписи PATH автоматически разворачиваются с индикатором количества совпадений.
- **Экспорт** — Экспорт переменных в форматы PowerShell, Batch или файл .env для совместной работы или восстановления системы.
- **Повышение UAC** — Автоматическое повышение прав администратора для редактирования системных переменных.
- **Мгновенная рассылка** — Изменения немедленно рассылаются системе через `WM_SETTINGCHANGE`.

## 100% Офлайн · Полностью бесплатно

EnvStudio работает **полностью офлайн, без телеметрии и сбора данных.** Ваши настройки остаются исключительно на вашем компьютере.

Все функции **полностью бесплатны**, без рекламы и платных ограничений. В будущем может быть добавлена возможность пожертвований, но никакие функции не будут ограничены.

Подробнее в нашей [Политике конфиденциальности](https://prunelab.net/ru/products/envstudio/privacy).

## Перед отправкой

Пожалуйста, сначала проверьте [существующие Issues](https://github.com/PruneLab/EnvStudio-Feedback/issues), чтобы избежать дубликатов.

При сообщении об ошибке укажите:
- **Версию EnvStudio** (в Настройках → О программе)
- **Версию Windows** (например, Windows 11 23H2)
- **Шаги для воспроизведения** проблемы
- **Ожидаемое поведение** vs **фактическое поведение**
- Скриншоты (если применимо)
